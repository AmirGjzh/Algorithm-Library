## How to use
1. Open the folder for the algorithm you need.
2. Most solutions are standalone source files. Compile/run with the usual toolchain:
   - C++: `g++ -std=c++17 file.cpp -O2 -o file && ./file`
   - Python: `python3 file.py`
   - Java: `javac File.java && java File`
3. Add problem-specific input where needed or modify the template for testing.
